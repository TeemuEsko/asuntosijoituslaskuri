# asuntosijoituslaskuri.fi Next.js MVP

## Käyttö

1. Pura zip.
2. Avaa kansio terminaalissa.
3. Aja:

```bash
npm install
npm run dev
```

4. Avaa:

```txt
http://localhost:3000
```

Mukana on frontend ja backend-reitti `/api/parse-listing`.

Huomio: Etuovi/Oikotie voi estää automaattisen haun. Silloin ilmoitustekstin parseri toimii varavaihtoehtona.
